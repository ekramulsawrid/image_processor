# Image Processor

Language: C++  
Starter program given by Professor [George Wolberg](http://www-cs.engr.ccny.cuny.edu/~wolberg/)

## Operations
  - Point Operations
    - Threshold
    - Clipping
    - Constrast Enhancement
    - Gamma Correction
    - Histogram Stretching
    - Histogram Mathcing
  - Neighborhood Operations
    - Error Diffusion
    - Blurring
    - Sharpening
    - Mediam Flitering
    - Convolution
